My Skills

I focus on providing high quality code that implement newest standards. A list of my skills and used technologies is presented below. I might not be an expert in all of them, but I work hard every day on broadening my knowledge and honing my skills.

HTML

I write semantic HTML5 code abiding W3C markup validation. My next goal is HTML accessibility.

CSS

I am proficient in RWD achieved with @media. I know and use flexbox and css-grid. My next goal is BEM naming convention.

JavaScript

I use newest standards of ES(6+). I know and use Ajax and OOP. I am working on using Clean Code principles (functional purity, clousers). I also use jQuery.

Git

For most of my projects I use GIT version control and github.com as my repository host. I prefer to use GIT console over other tools.

Bootstrap

I use Bootstrap mostly for it's responsive grid system and easy to implement reusable components like navs, forms, sliders, etc..

Sass

For a more advanced projects I prefer to use Sass over pure CSS. For managing my workflow I usually pair Sass with gulp.js.

Angular

I learned basics of Angular7 and TypeScript and made few small and one bigger project. Now I focus more on pure JavaScript and leave the choice of framework for my future employer.

React
